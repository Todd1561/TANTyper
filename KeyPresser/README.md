# TANTyper

Types the specified keys out to whatever window has focus in an endless loop.

## Examples
Type 'hello' in endless loop  
`tantyper.exe "h,e,l,l,o"`

Press the Capslock key in endless loop  
`tantyper.exe "{CAPSLOCK}"`
